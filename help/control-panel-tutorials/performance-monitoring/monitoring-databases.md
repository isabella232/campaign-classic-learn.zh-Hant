# 監控資料庫

您的促銷活動例項的所有資產、工作流程和資料都儲存在資料庫中。 資料庫達到最大容量可能會導致您的例項發生問題（無法登入、傳送電子郵件等）。 因此，必須定期監視實例的資料庫以確保最佳效能。
